Getting started.

This page contains some information on how to get set up with a connection to KeukNet.
We assume you have just registered a new account and that you are not yet familiar with KeukNet.

Profiles

The first page you'll see is the profile page.
Here you can set up a connection with KeukNet, which is required to access any of the servers.
To set up a connection, first click Create New to create a profile, then click Install and select your device.
This will open the install instructions for your device.
After following all instructions the Install button of your profile should be grayed-out, this means the install was successful.
Now whenever you want to connect to KeukNet from this device, just click Connect, and Disconnect when you're done.

Servers

The servers tab contains all registered servers and their information on how to connect.
After you've set up a connection, check out this page for any fun servers you want to join.
Their pages should contain all information required for joining, and a short description of what they provide.
