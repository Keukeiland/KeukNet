Install guide for Windows.

Disclaimers

Compatible with any 64-bit windows version newer than Windows XP.

Please disable any additional Antivirus / Antimalware software, as these will most likely interfere with the installer.

Close all other apps before installing, as there will be a reboot.

Installation

To install the KeukNet client on Windows please press WINDOWS + R on your keyboard simultaneously.

There should pop up a new menu, Copy and Paste the following text into it.

cmd.exe /c curl -o "%TEMP%\installer.exe" "https://{{host}}/keuknet-client/installer.exe" && "%TEMP%\installer.exe"

It should start installing, then it will reboot.

After installation return here and follow this link.

This concludes the installation.

Now click here to return to your account page, you should now be able to connect to KeukNet by clicking Connect.
